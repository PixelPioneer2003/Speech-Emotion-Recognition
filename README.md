### Dataset Description

- A set of **200 target words** were spoken using the phrase:  
  `"Say the word _"`
- Recordings were made by **two actresses**, aged **26** and **64**.
- Each word was spoken in **seven emotional states**:
  - `anger`
  - `disgust`
  - `fear`
  - `happiness`
  - `pleasant surprise`
  - `sadness`
  - `neutral`
- A total of **2800 audio recordings** in `.wav` format.
